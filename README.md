# node.js 데모

> ## dependencies
> **body-parser** : request 파라미터값을 쉽게 사용  
> **dotenv** : env 파일 사용하는데 편리  
> **ejs** : viewEngine  
> **express, express-session** : framework  
> **mongodb** : DBMS  
> **multer** : 파일 업로드 구현에 추가한 lib  
> **nodemon** : 소스 수정시 자동으로 server 재시작 해줌  
> **passport, passport-local** : 세션 관리 lib
